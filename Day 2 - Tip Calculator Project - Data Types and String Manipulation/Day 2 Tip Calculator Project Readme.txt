Day 2 - Tip Calculator Project Readme

Description:
This program calculates the amount each person should pay when splitting a bill. The user is prompted to input the total bill, the percentage tip they'd like to give, and the number of people to split the bill. The program then calculates the total bill including the tip and divides it by the number of people to determine how much each person should contribute.

How to Run:
Run the script:python main.py
Enter the total bill: The program will ask for the amount of the bill.
Enter the tip percentage: Choose from 10%, 12%, or 15%.
Enter the number of people: Specify how many people are sharing the bill.
Get the result: The program will calculate and display the amount each person should pay.

How It Works (Code Overview):
User Input:
The program first asks for the total bill amount.
Then, it prompts for the tip percentage (10%, 12%, or 15%).
Finally, the program asks how many people will split the bill.

Calculation:
The program calculates the total bill including the tip and then divides it by the number of people to determine how much each person should pay.

Output:
The program outputs the calculated amount that each person should pay.

Key Features:
Simple bill splitting based on user input.
Allows for different tip percentages (10%, 12%, 15%).
Rounds the result to 2 decimal places for clarity.
