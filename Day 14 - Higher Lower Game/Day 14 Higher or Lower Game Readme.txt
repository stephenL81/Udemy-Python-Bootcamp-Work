Day 14 - Higher or Lower Project Readme

Description:
This is a simple "Higher or Lower" game where the player must guess which celebrity has a higher follower count. The game randomly selects two people from a dataset and asks the user to choose who has more followers. If the player guesses correctly, they score a point and continue; if they guess wrong, the game ends.

How to Play:
The game displays two celebrities along with their name, description, and country.
The player types 'A' or 'B' to guess who has more followers.
If correct, the score increases, and a new opponent is chosen.
If incorrect, the game ends and displays the final score.

Key Features:
Uses random selection to pick celebrities.
Keeps track of already used celebrities to avoid repetition.
Implements a looping gameplay until the user guesses incorrectly.
