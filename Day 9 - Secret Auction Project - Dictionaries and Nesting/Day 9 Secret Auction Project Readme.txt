Day 9 - Secret Auction Project Readme

Description:
This program simulates a secret auction where multiple participants can enter their bids. The highest bidder is determined at the end, and their name is displayed as the winner.

How to Use:

Run the script: python main.py
Enter the name of the bidder.
Enter their bid amount.
Choose whether to add another bid or end the auction.
When all bids are entered, the program determines and displays the highest bidder.

How It Works (Code Overview):
art.py → Contains ASCII art for the auction logo.
main.py → Handles user input, stores bids in a dictionary, and determines the highest bidder.

Key Features:

Uses a dictionary to store bidders' names and their corresponding bids.
Clears the screen between bids to keep the auction secret.
